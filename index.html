The stateful widget is mounted and the state is set with person: unloadedAsyncState().

The async widget checks what the state is of s0.person. Because we set it to unloadedAsyncState(), the async widget does nothing.

Within the code, the state of s0.person is checked and based on the kind, the appropriate view is rendered.

When clicking on the 'Load person' button, the state is updated and person is set to loading with loadingAsyncState(() => fetchPerson()).

At this point, our mutated state is passed to the children of any and the async widget checks what s0.person is. It sees that s0.person is mutated to loading and fires the fetch.

After the fetch is complete, successful or not, the state is mutated with the result.

The state is updated with the output of the async widget.

The children of stateful are rerendered, and our mkWidget recieves the updated state. Inside the widget, it checks if s0.person is loaded, which at this point is, and renders the result.
